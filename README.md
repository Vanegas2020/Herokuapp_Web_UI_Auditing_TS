## Setup

```bash
# 1. Install Node.js (>= 24.0.0)

# 2. Install dependencies
npm install

# 3. Install Playwright browsers
npx playwright install --with-deps chromium

# 4. Configure environment variables
cp .env.example .env
# Edit .env and fill in your values (URLs, credentials, etc.)
```

## Running the audit

```bash
npm test
```

## Options

| Variable | Default | Description |
|----------|---------|-------------|
| `HEADED` | `false` | Set `true` to see the browser window during the audit |
| `SLOWMO` | `0` | Milliseconds to slow down each action (useful for debugging) |
| `CI` | `false` | Set `true` on CI environments |
| `WORKERS` | `1` | Number of parallel browser workers (default is 1 for stability) |
| `PAGE_URL_*` | *(recorded URL)* | Override any page URL — see `.env.example` for variable names |
| `AUTH_*_USERNAME` | — | Login username for pages that require authentication |
| `AUTH_*_PASSWORD` | — | Login password for pages that require authentication |
